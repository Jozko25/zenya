## üîç What to Look For in a Good Loop

When selecting files on Pixabay/Freesound:

### ‚úÖ Good Indicators:
- File name contains "loop" or "seamless"
- Description mentions "loopable" or "seamless loop"
- Duration: 30 seconds to 2 minutes (longer is better)
- Sample rate: 44.1kHz or 48kHz
- File format: MP3 or WAV
- License clearly states "CC0" or "Royalty-Free"

### ‚ùå Avoid:
- Very short files (< 10 seconds) - loop will be obvious
- Low quality (< 44.1kHz)
- Files with fade-in/fade-out (makes looping hard)
- Files that say "one-shot" or "single"
- Unclear licensing

class MeditationAudioPlayer: NSObject, ObservableObject {
    private var audioPlayer: AVAudioPlayer?
    
    func setupAudio(soundType: String) {
        let fileName = mapSoundToFile(soundType)
        
        guard let url = Bundle.main.url(
            forResource: fileName, 
            withExtension: "mp3"
        ) else {
            print("Audio file not found: \(fileName)")
            return
        }
        
        do {
            // Configure audio session for background playback
            try AVAudioSession.sharedInstance().setCategory(
                .playback, 
                mode: .default
            )
            try AVAudioSession.sharedInstance().setActive(true)
            
            // Create player
            audioPlayer = try AVAudioPlayer(contentsOf: url)
            audioPlayer?.numberOfLoops = -1  // Loop forever
            audioPlayer?.prepareToPlay()
            
        } catch {
            print("Audio setup error: \(error)")
        }
    }
    
    func play() {
        audioPlayer?.play()
    }
    
    func pause() {
        audioPlayer?.pause()
    }
    
    func stop() {
        audioPlayer?.stop()
        audioPlayer?.currentTime = 0
    }
    
    private func mapSoundToFile(_ soundType: String) -> String {
        switch soundType.lowercased() {
        case "white noise": return "white_noise_loop"
        case "brown noise": return "brown_noise_loop"
        case "pink noise": return "pink_noise_loop"
        case "forest rain": return "rain_loop"
        case "ocean waves": return "ocean_waves_loop"
        case "thunderstorm": return "thunderstorm_loop"
        case "crackling fire": return "fire_crackling_loop"
        case "mountain stream": return "stream_loop"
        default: return "white_noise_loop"
        }
    }
}
